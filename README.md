# ShipBy

  ShipByは、Shopifyの注文に対して「出荷日（ship-by）」を自動で算出し、注文に保存するアプリです。
  出荷リードタイム、配送可能エリア、休業日をもとに、正確な出荷日を一貫して管理できます。

  ## 主な機能
  - 出荷日を自動計算して注文に保存
  - 配送可能エリアごとのリードタイム設定
  - 休業日を考慮した出荷日調整
  - 必要に応じてタグ付け（任意）
